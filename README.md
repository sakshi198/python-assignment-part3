# Python Assignment - Part 3: Product Explorer & Logger

This project demonstrates file handling, API integration, exception handling, and logging using Python.

## Tasks Covered

### Task 1: File Handling
- Wrote and appended notes to a file
- Read file content and performed keyword search

### Task 2: API Integration
- Fetched product data from DummyJSON API
- Filtered and sorted products
- Retrieved category-based products
- Performed POST request

### Task 3: Exception Handling
- Safe division function
- Safe file reading
- Robust API calls with error handling
- Input validation loop

### Task 4: Logging
- Logged errors with timestamps
- Handled connection and HTTP errors
- Displayed log file content
